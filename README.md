# CPBL Hitter Projection System

This project develops a preseason projection system for CPBL hitters using historical player statistics from Baseball Reference.

The goal is to predict:

- Home Runs (HR)
- On-Base Percentage (OBP)
- Slugging Percentage (SLG)
- OPS

## Methods

- Exploratory Data Analysis
- Feature Engineering
- Rolling Cross Validation
- Lasso Regression
- Ridge Regression
- Elastic Net Regression

## Results

The final models were used to generate 2026 CPBL hitter projections.

Top projected hitters included:

- Steven Moya
- An Ko Lin
- Kungkuan Giljegiljaw
- Yu Chang
